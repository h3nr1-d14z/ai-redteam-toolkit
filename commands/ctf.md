Solve CTF challenge: $ARGUMENTS

1. **Setup**: create `ctf/<platform>/<challenge>/` with `files/`
2. **Analyze** by category:
   - Web: source, SQLi, XSS, SSTI, SSRF, LFI, command injection
   - Pwn: checksec, find vuln (BOF/format string/UAF), build exploit
   - RE: decompile, understand algorithm, extract flag
   - Crypto: identify cipher, find implementation flaws, decrypt
   - Forensics: file analysis (binwalk/exiftool), memory/network forensics, stego
3. **Solve**: write `solve.py`, get flag
4. **Writeup**: create `writeup.md` with description, solution steps, flag, key takeaway
